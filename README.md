# Walk This Way 🏃

This is an arcade theme inspired circular walking route generator that turns your daily step goals into a gamified adventure.

### 🕹️ Features
- **Circular Routes**: Automatically generates a walking path that starts and ends at your current location.
- **Gamified Progress**: Checkpoints and celebrations integrated directly onto your route polyline.
- **Arcade Aesthetic**: A retro-style interface that makes fitness feel like a high-score challenge.
- **Real-Time Tracking**: Live GPS tracking and proximity detection for a seamless walking experience.

### 🚀 Tech Stack
- **Framework**: React Native / Expo
- **Maps**: Google Maps Directions & JavaScript SDK
- **Styling**: Custom arcade-style components and "Saira Stencil One" typography
